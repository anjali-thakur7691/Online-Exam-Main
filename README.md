### *A Smart & User-Friendly Online Examination Platform*

A modern **Online Exam Management System** built with **Python** and **Streamlit**. This application allows administrators to create exams and students to take tests online with automatic result generation.

---

## ✨ Features

- 🔐 Secure Login & Registration
- 👨‍🎓 Student Dashboard
- 👨‍💼 Admin Dashboard
- 📝 Create & Manage Exams
- ❓ Add Multiple Choice Questions
- ⏱️ Timed Online Tests
- 📊 Instant Result Generation
- 📈 Score & Performance Tracking
- 💾 Database Support

---

## 🛠️ Tech Stack

- Python
- Streamlit
- SQLite
- Git & GitHub

## 🚀 How to Run

```bash
git clone https://github.com/anjali-thakur7691/Online-Exam-Main.git
cd Online-Exam-Main
pip install -r requirements.txt
streamlit run app.py
```
